InduDoc Transformer Documentation
==================================

**InduDoc Transformer** is a Python package for processing and extracting structured data 
from industrial PDF documents, particularly electrical diagrams and documentation.

Features
--------

* Extract structured tags and components from industrial electrical diagrams
* Parse hierarchical tag systems with configurable separators
* Track connections between components (devices, terminals, cables)
* Support for multiple document formats via plugin system
* Export to various formats including AML and databases
* GUI interface for visualization and interaction
